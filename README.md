# 🛍️ Dangi Suraj Sari Center

A premium, modern, and responsive fashion website built for showcasing **saree, lehenga, and fabric collections**.
This project is designed to represent a real sari shop and provide users with a smooth and elegant browsing experience.

---

## 🌐 Live Website

🔗 https://dangi-suraj-sari-center.vercel.app/

---

## 📌 About the Project

**Dangi Suraj Sari Center** is a fashion showroom website based in **Soyat Kalan, India**.
The website highlights:

* Bridal Sarees
* Party Wear Sarees
* Lehenga Collection
* Men’s Fabric (non-ready-made cloth)

It is built with a focus on:

* Clean UI
* Smooth animations
* SEO optimization
* Mobile responsiveness

---

## 🚀 Features

✨ Fully Responsive Design (Mobile + Desktop)

---

## 🧭 Pages & Routes

* `/` → Home
* `/collection` → All Products
* `/category/saree` → Saree Collection
* `/category/lehenga` → Lehenga Collection
* `/category/men` → Men Fabric Collection
* `/product/:id` → Product Details

## ⚙️ Installation & Setup

Clone the repository:

```bash
git clone https://github.com/your-username/Dangi-Suraj-Sari-Center.git
```

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npm start
```

---

## 🚀 Deployment

This project is deployed using **Vercel**.

To deploy:

```bash
npm run build
```

Then upload to Vercel.
